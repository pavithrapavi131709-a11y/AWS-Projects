# WordPress on AWS Lightsail

This project explains the deployment of a WordPress website using AWS Lightsail.

## Tools Used
- AWS Lightsail
- WordPress
- Apache
- MariaDB
- PHP

## Steps
- Create Lightsail instance
- Select WordPress blueprint
- Launch instance
- Access using Public IP
- Connect using SSH
- Get admin password
- Login to WordPress dashboard
- Configure website
- Install theme and plugins
- Add pages and products

## Outcome
Successfully deployed a WordPress website on AWS Lightsail.

## Learning
- Cloud hosting
- WordPress setup
- AWS basics
- Linux server management
